# MVP Scope

## MVP Use Case

The MVP demonstrates an AI-assisted Oracle → BigQuery modernization workflow.

## Available Today / MVP Focus

- Schema discovery from uploaded metadata
- Migration assessment
- Source-to-target mapping generation
- Basic SQL transformation
- Validation plan generation
- Migration planning
- Human review workflow concept
- Sample generated artifacts

## Coming Soon

## MVP Success Criteria

The MVP should prove that one complete modernization workflow can generate useful, reviewable outputs:

1. Assessment report
2. Source-to-target mapping
3. BigQuery DDL
4. Converted SQL
5. Validation report
6. Cutover runbook
7. Executive scorecard
