# Demo Script

## 30-Second Pitch

Modernization AI Lab is an AI-native Enterprise Modernization Operating System that helps enterprises modernize legacy data platforms to cloud and AI-ready platforms using specialized AI agents, Enterprise Context Graph and human-in-the-loop governance.

## 3–5 Minute Demo Flow

### 1. Problem

Enterprise modernization fails because discovery, mapping, transformation, validation and cutover remain manual and fragmented.

### 2. Platform

Show the homepage and explain the modernization lifecycle:

Discover → Assess → Design → Transform → Validate → Deploy → Govern → Optimize

### 3. Maturity Assessment

Show the maturity assessment and explain how organizations benchmark modernization readiness.

### 4. AI Agent Catalog

Show the agent catalog and explain the MVP agents versus future platform vision.

### 5. Oracle → BigQuery Use Case

Walk through:

- Upload metadata
- Generate assessment
- Generate mappings
- Generate DDL and SQL
- Generate validation report
- Generate cutover runbook

### 6. Moat

Explain the Enterprise Context Graph and human-in-the-loop governance.

### 7. Close

Ask for design partner feedback, Google architecture guidance and pilot opportunities.
