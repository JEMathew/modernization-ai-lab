# Seller Playbook

## Two-Minute Explanation

Modernization AI Lab helps enterprises modernize legacy data platforms using governed AI agents. It automates discovery, assessment, mapping, transformation, validation and cutover planning while keeping humans in control of critical decisions.

## Ideal Customer Profile

- Enterprises with large Oracle, SQL Server, Teradata, Hadoop or Informatica estates
- Organizations pursuing cloud migration
- Enterprises building AI-ready data foundations
- CIO/CDO/CTO teams facing modernization backlog

## Key Differentiators

- Enterprise Context Graph
- Multi-agent modernization workflows
- Human-in-the-loop governance
- Reviewable modernization artifacts
- Full lifecycle modernization, not just code conversion

## Common Objections

| Objection | Response |
|---|---|
| Is this just a wrapper? | No, the moat is context graph + workflows + governance. |
| Can AI be trusted? | Critical outputs are reviewed, versioned and auditable. |
| Is this only for one database? | MVP starts with Oracle → BigQuery, platform vision expands across ecosystems. |
