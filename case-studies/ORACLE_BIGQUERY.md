# Case Study: Oracle To BigQuery

Synthetic example for product demonstration.

## Problem

Legacy Oracle reporting estate with complex SQL, manual validation and high SME dependency.

## Approach

- Discover schema and metadata
- Assess complexity
- Generate mappings
- Convert DDL and SQL
- Generate validation evidence
- Prepare cutover runbook

## Outcome

Illustrative outcome: faster assessment, better validation coverage and reusable migration playbook.
